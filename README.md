# pi-config

## Setup Hotspot

1. wget https://raw.githubusercontent.com/danielr18/pi-config/master/setup_ap.sh
2. sudo chmod u+x setup_ap.sh
3. sudo ./setup_ap.sh

## Start WiFi Hotspot

`sudo ~/.digitalpignage/start_wifi_ap.sh`

## Start WiFi Client

`sudo ~/.digitalpignage/start_wifi_client.sh`
